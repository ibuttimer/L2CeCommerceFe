A demonstartion full stack E-commerce application with:

  • login/logout
  • product catalog
  • shopping cart
  • checkout
  • card payments

It is based on the E-commerce application from the Full Stack: Angular and Java Spring Boot course on Udemy, and has been extended beyond the basic functionality outlined in the course.

Note: This is not a retail site, do not enter valid payment card or personal details. Please use Stripe test card numbers.
Tech stack

The application consists of:

  • Spring Boot application with:
    • Spring Data JPA providing REST APIs
    • Stripe for card payment processing
    • JWT, oauth2, OpenID Connect provided by okta
  • Database supported:
    • MySQL
    • PostgreSQL
  • Angular frontend application
